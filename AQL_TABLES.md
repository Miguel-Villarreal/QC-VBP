## How to Use These Tables

1. **Look up the code letter** in Table I using the lot size and inspection level (e.g., lot of 3,000 at Level II = code K)
2. **Find the sample size** from the code letter (K = 125 units)
3. **Find the Ac/Re numbers** in Table II-A at the intersection of the code letter and AQL level (K at AQL 2.5 = Ac 7, Re 8)
4. **Decision**: If non-conforming units found <= Ac, the lot PASSES. If >= Re, the lot FAILS.

### Example

- Lot size: 3,000 units
- Inspection level: II (general)
- AQL: 2.5%
- Table I: 1201-3200 at Level II = code letter **K**
- Sample size: **125 units**
- Table II-A: K at 2.5 = **Ac 7, Re 8**
- Inspect 125 units. If 7 or fewer are non-conforming: PASS. If 8 or more: FAIL.

---

## Notes

- The accept numbers used in the standard are: 0, 1, 2, 3, 5, 7, 10, 14, 21
- The reject number is always Ac + 1
- Level II is the most commonly used general inspection level
- AQL range covered: 0.065 to 6.5
- This table covers Normal inspection severity only (Tightened and Reduced plans exist but are not included here)
- Arrow cells from the original standard are resolved by following the arrow to the referenced plan's Ac/Re values
